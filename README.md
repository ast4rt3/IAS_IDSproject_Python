# Process Manager 🚀  
A simple Python-based process manager with a watchlist feature.  
by:  Lawrence Heras
## Features  
- 🔍 **Search** for running processes  
- ❌ **Kill** selected processes  
- 🚫 **Blocklist** processes  
- 👀 **Watchlist** notification for running/stopped processes
